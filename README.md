# Ứng dụng Hỗ trợ Nông dân Đồng Bằng Sông Cửu Long

## Tính năng chính

### 1. Hệ thống Dự đoán Xâm nhập Mặn
- Dự báo độ mặn 7-14 ngày
- Biểu đồ trực quan, dễ hiểu
- Bản đồ vùng bị ảnh hưởng

### 2. Cộng đồng & Chia sẻ Kinh nghiệm
- Đăng bài chia sẻ kinh nghiệm
- Hệ thống điểm uy tín
- Like, comment, tương tác
- Phân loại theo chủ đề
- Top thành viên xuất sắc

### 3. Chợ Nông sản & Thiết bị
- Mua bán thiết bị nông nghiệp
- Giống cây trồng chịu mặn
- Máy móc, phân bón, vật tư
- Liên hệ trực tiếp người bán
- Đánh giá qua điểm uy tín

### 4. Đầu tư & Hợp tác
- Kêu gọi vốn cho dự án
- Kết nối nhà đầu tư
- Hợp tác doanh nghiệp
- Liên kết tổ chức khoa học

## Công nghệ

- **Framework**: React + TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: Shadcn/UI
- **Charts**: Recharts
- **Icons**: Lucide React
- **Build**: Vite

## Dữ liệu Demo (chưa chạy model)

Hệ thống sử dụng mock data thực tế:
- 48,500+ nông dân tham gia
- 125,000 ha diện tích canh tác
- 13 tỉnh Đồng Bằng Sông Cửu Long
- Dự án đầu tư thực tế
- Sản phẩm nông nghiệp đa dạng

## Khởi chạy

```bash
# Cài đặt dependencies
npm install

# Chạy development server
npm run dev

# Build production
npm run build
```
